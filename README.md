## 🎨 Color Palette & Distribution

| Rol | Hex | ¿Dónde se usa? |
|-----|-----|----------------|
| 60% Base | #F0F3F7 | Se utiliza como fondo general de la aplicación, contenedores principales y espacios vacíos. Este color permite que el contenido destaque y mejora la legibilidad. |
| 30% Secondary | #112331 | Se aplica en la sidebar y en la estructura de navegación principal. Ayuda a definir la jerarquía visual y separar claramente las zonas de interacción. |
| 10% Accent | #34495E | Se reserva para botones, gráficos principales y elementos interactivos. Su uso es limitado para que realmente destaque en acciones importantes. |

## 🧠 Contexto general

La distribución de estos colores sigue la regla 60-30-10 para mantener equilibrio visual dentro de la interfaz. El uso de tonos claros como base permite reducir la fatiga visual, mientras que los tonos oscuros ayudan a estructurar la navegación. El color de acento se utiliza de forma controlada para guiar al usuario hacia las acciones más importantes.

Además, todos los colores fueron validados con herramientas de contraste para cumplir con los estándares de accesibilidad WCAG 2.1. Esto asegura que la interfaz no solo sea visualmente agradable, sino también usable para personas con diferentes capacidades visuales.
